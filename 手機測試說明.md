# 手機測試網頁說明

## 方法一：使用本地網路 IP（推薦）

### 步驟 1：修改 Flask 配置
已修改 `app.py`，現在會監聽所有網路介面（`host='0.0.0.0'`）

### 步驟 2：獲取電腦的 IP 地址

**Windows:**
```bash
ipconfig
```
找到 "IPv4 地址"，通常是 `192.168.x.x` 或 `10.x.x.x`

**或者使用 PowerShell:**
```powershell
Get-NetIPAddress -AddressFamily IPv4 | Where-Object {$_.InterfaceAlias -notlike '*Loopback*'}
```

### 步驟 3：啟動 Flask 應用
```bash
python app.py
```

### 步驟 4：在手機上訪問
1. 確保手機和電腦連接到**同一個 WiFi 網路**
2. 在手機瀏覽器中輸入：
   ```
   http://你的電腦IP:5000
   ```
   例如：`http://192.168.1.100:5000`

### 步驟 5：允許防火牆（如果需要）
如果無法訪問，Windows 可能會提示允許防火牆：
- 點擊「允許訪問」
- 或手動在 Windows 防火牆中添加 Python 的例外

## 方法二：使用 ngrok（適合需要 HTTPS 的情況）

### 安裝 ngrok
1. 下載：https://ngrok.com/download
2. 解壓縮到任意位置

### 使用步驟
1. 啟動 Flask 應用（`python app.py`）
2. 開啟新的終端，運行：
   ```bash
   ngrok http 5000
   ```
3. ngrok 會顯示一個公開的 URL，例如：
   ```
   https://xxxx-xx-xx-xx-xx.ngrok.io
   ```
4. 在手機瀏覽器中訪問這個 URL（不需要同一個 WiFi）

**優點：**
- 不需要同一個 WiFi
- 提供 HTTPS（相機功能需要）
- 可以分享給其他人測試

**缺點：**
- 免費版有連接數限制
- URL 每次啟動都會改變

---

## 方法三：使用手機熱點

1. 開啟手機熱點
2. 電腦連接到手機熱點
3. 使用電腦的 IP 地址訪問（通常是 `192.168.43.x`）

---

## 注意事項

### 1. 相機功能需要 HTTPS
- 瀏覽器要求相機功能必須在 HTTPS 環境下使用
- 本地 HTTP 可能無法使用相機
- 解決方案：
  - 使用 ngrok（提供 HTTPS）
  - 或使用手機瀏覽器的開發者模式（可能有限制）

### 2. 防火牆設置
如果無法訪問，檢查：
- Windows 防火牆是否允許 Python
- 路由器防火牆設置
- 防毒軟體的網路保護

### 3. 網路連接
- 確保手機和電腦在同一網路
- 檢查 IP 地址是否正確
- 嘗試 ping 測試：在手機上 ping 電腦 IP

### 4. 端口衝突
如果 5000 端口被占用，可以修改：
```python
app.run(host='0.0.0.0', port=8080, debug=True)
```
然後訪問：`http://你的IP:8080`

## 快速測試腳本

創建一個 `get_ip.py` 來快速獲取 IP：

```python
import socket

def get_local_ip():
    s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
    try:
        # 連接到一個遠程地址（不會實際發送數據）
        s.connect(('8.8.8.8', 80))
        ip = s.getsockname()[0]
    except Exception:
        ip = '127.0.0.1'
    finally:
        s.close()
    return ip

if __name__ == "__main__":
    ip = get_local_ip()
    print(f"\n{'='*50}")
    print(f"你的本地 IP 地址是: {ip}")
    print(f"在手機瀏覽器中訪問: http://{ip}:5000")
    print(f"{'='*50}\n")
```

運行：`python get_ip.py`
